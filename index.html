Sundial

Sundial is a self-hosted, open-source cron job monitoring and management system that provides real-time bi-directional communication between the user and registered crontabs on one or multiple nodes. Built with Node.js ...

For the developer, Sundial provides both a UI and CLI to abstract away the pain of monitroing and managing cron jobs. Furthermore, Sundial notifies of failures through webhooks or Slack and gathers all cron job outputs and error logs into the dashboard.
